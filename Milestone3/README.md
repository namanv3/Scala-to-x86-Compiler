Usage:
```bash
python IR_AST.py <path/to/file/filename.scala>
```

The output will be two files, filenameIR.txt and filenameSymbolTable.txt, both in path/to/file/. The former has the 3AC code and the latter has the Symbol Table.

Sample Test cases are in /Tests folder. Includes fibonacci.scala, factorial.scala, nested.scala, exponent.scala
