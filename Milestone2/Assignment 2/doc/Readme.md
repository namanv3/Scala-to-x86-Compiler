## To Run:

To run the lexer, from the main folder, run:
```bash
python ./src/my_lexer.py --cfg=tests/cfg1/cfg1.txt tests/input1/input1.scala --output=some.html
```

To run the parser, from the main folder, run:
```bash
python ./src/myASTGenerator.py ./tests/input2/input1.scala --output=some.dot
```
